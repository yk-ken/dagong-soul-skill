# 代价计算器 Prompt

本文件定义维权代价计算器的计算维度、公式和输出格式。cost_calculator 为用户提供各维权路径的量化对比，帮助用户做出理性决策。

---

## 一、计算维度

### 1. 经济收益

用户通过维权可能获得的经济利益。

| 收益项 | 计算方式 | 适用条件 |
|--------|---------|---------|
| 经济补偿金（N） | 工作年限 × 月平均工资 | 合法解除或协商解除 |
| 违法解除赔偿金（2N） | 工作年限 × 月平均工资 × 2 | 公司违法解除 |
| 试用期工资差额 | (转正工资 × 80% - 实发工资) × 月数 | 试用期工资低于80% |
| 未签合同双倍工资差额 | 月工资 × 未签月数（最长11个月） | 超过1个月未签合同 |
| 加班费 | 按法定标准（平时1.5倍/周末2倍/节假日3倍） | 存在加班事实且有证据 |
| 未休年假折算 | 日工资 × 未休天数 × 200% | 有未休法定年假 |
| 代通知金 | 1个月工资 | 适用劳动合同法第40条情形 |

### 2. 时间成本

各维权路径所需的时间投入。

| 路径 | 预计总时间 | 用户需投入时间 | 说明 |
|------|-----------|--------------|------|
| 协商 | 1-2 周 | 2-5 小时 | 主要是约谈和文件往来 |
| 仲裁 | 45-60 天 | 20-40 小时 | 准备材料、等待受理、参加开庭 |
| 一审诉讼 | 3-6 个月 | 40-80 小时 | 立案、举证、开庭、等判决 |
| 二审诉讼 | 额外 3 个月 | 20-40 小时 | 上诉、开庭、等判决 |
| 强制执行 | 2-6 个月 | 10-20 小时 | 申请、提供线索、等待执行 |

### 3. 机会成本

维权期间影响找新工作的时间窗口。

| 因素 | 影响程度 | 说明 |
|------|---------|------|
| 精力分散 | 中-高 | 维权需投入大量精力，影响求职准备 |
| 心理状态 | 中-高 | 维权过程可能影响面试表现和心态 |
| 时间占用 | 低-中 | 协商影响小，仲裁/诉讼需请假开庭 |
| 空窗期延长 | 低-中 | 视维权投入时间而定 |

**机会成本估算：**
```
机会成本 = 月预期工资 × 维权占用月份数 × 精力分散系数

精力分散系数：
  协商：0.1（影响很小）
  仲裁：0.3（有一定影响）
  诉讼：0.5（影响较大）
```

### 4. 背调风险

维权行为对未来求职的潜在影响。

| 维权路径 | 背调风险等级 | 说明 |
|---------|------------|------|
| 协商解决 | 极低 | 无公开记录，不影响背调 |
| 劳动仲裁 | 低-中 | 仲裁记录不公开，但前雇主可能在背调时提及 |
| 法院诉讼 | 中 | 判决文书可能上传至裁判文书网（但劳动争议案件通常不公开） |
| 社交媒体曝光 | 高 | 可能被前雇主或背调公司搜索到 |

**风险缓解措施：**
- 仲裁/诉讼记录通常不会出现在常规背调中
- 劳动争议判决文书公开范围有限
- 维权是合法权利，优质企业通常不会因此拒绝录用
- 可在面试中正面表述为"合法维护自身权益"

### 5. 心理成本

长期对抗的精神压力评估。

| 维权路径 | 心理压力等级 | 持续时间 | 说明 |
|---------|------------|---------|------|
| 协商 | 低 | 1-2 周 | 过程短，不确定性低 |
| 仲裁 | 中 | 2-3 个月 | 需要准备材料、出庭，有一定压力 |
| 诉讼 | 高 | 6-12 个月 | 周期长，不确定性高，精神消耗大 |

**心理成本量化（仅供参考）：**
```
心理成本指数：
  协商：1-3（轻微不适）
  仲裁：4-6（中度压力）
  诉讼：7-9（高度压力）

影响维度：
  - 日常情绪波动
  - 睡眠质量
  - 家庭关系
  - 工作表现
```

## 二、计算公式

### 1. 协商预期值

```
协商预期值 = 预期赔偿金额 × 达成概率 / 时间投入（小时）

其中：
  预期赔偿金额 = 目标协商金额（基于N或2N）
  达成概率 = 基于案件分析（通常协商成功概率较高）
  时间投入 = 预计投入的维权时间

示例：
  预期赔偿 = 8,000 元
  达成概率 = 80%
  时间投入 = 5 小时
  协商预期值 = 8,000 × 80% / 5 = 1,280 元/小时
```

### 2. 仲裁预期值

```
仲裁预期值 = 预期赔偿金额 × 胜诉率 / (时间投入 × 机会成本系数)

其中：
  预期赔偿金额 = 主张金额（通常为2N + 其他）
  胜诉率 = 基于案件分析的胜诉概率
  时间投入 = 预计投入的维权时间（小时）
  机会成本系数 = 1 + 精力分散系数

示例：
  预期赔偿 = 8,000 元
  胜诉率 = 70%
  时间投入 = 30 小时
  机会成本系数 = 1.3
  仲裁预期值 = 8,000 × 70% / (30 × 1.3) = 143.6 元/小时
```

### 3. 诉讼预期值

```
诉讼预期值 = 预期赔偿金额 × 胜诉率 / (时间投入 × 机会成本系数) × 折现系数

其中：
  折现系数 = 1 / (1 + 月利率 × 预计月数)
  （反映远期收益的时间价值，月利率按同期银行存款利率近似）

示例：
  预期赔偿 = 8,000 元
  胜诉率 = 75%
  时间投入 = 60 小时
  机会成本系数 = 1.5
  预计月数 = 6 个月
  折现系数 ≈ 0.99（6个月时间价值影响很小）
  诉讼预期值 = 8,000 × 75% / (60 × 1.5) × 0.99 = 66 元/小时
```

### 4. 综合预期值条形图数据

```
综合预期值 = 经济收益 × 权重1 + 时间成本得分 × 权重2 + 精力成本得分 × 权重3
           + 背调风险得分 × 权重4 + 心理成本得分 × 权重5

权重建议：
  经济收益：0.35
  时间成本：0.20
  精力成本：0.15
  背调风险：0.15
  心理成本：0.15

各项得分（1-10分，10为最优）：

经济收益得分：
  协商：赔偿金额 × 达成概率 / 满分金额 × 10
  仲裁：赔偿金额 × 胜诉率 / 满分金额 × 10
  诉讼：赔偿金额 × 胜诉率 / 满分金额 × 10

时间成本得分（越短越高）：
  协商：10（1-2周）
  仲裁：6（2-3月）
  诉讼：3（6-12月）

精力成本得分（越少越高）：
  协商：9（2-5小时）
  仲裁：5（20-40小时）
  诉讼：2（60-80小时）

背调风险得分（越低越高）：
  协商：10（几乎无风险）
  仲裁：7（低风险）
  诉讼：5（中等风险）

心理成本得分（越低越高）：
  协商：9
  仲裁：5
  诉讼：2
```

## 四、计算原则

### 核心原则

1. **先给理性分析，再尊重用户选择**
   - 计算器首先给出各路径的客观量化对比
   - 明确标注推荐路径和推荐理由
   - 但最终选择权完全在用户

2. **数据驱动**
   - 所有数字基于用户提供的案情数据
   - 概率基于 legal_judgment 的胜诉概率评估
   - 时间估算基于法定时限和实际经验

3. **透明计算**
   - 每个数字附带计算过程
   - 概率评估附带理由
   - 不确定性明确标注

4. **不夸大收益**
   - 使用概率加权后的预期收益
   - 考虑机会成本和心理成本
   - 提醒实际结果可能与预期不同

### 计算触发条件

cost_calculator 在以下情况下触发计算：

1. 用户选择"还没想好"时 → 自动输出完整对比
2. 用户询问"划算吗""值不值得"时 → 输出当前路径的详细分析
3. 用户要求对比各方案时 → 输出完整对比
4. strategy_builder 生成策略时 → 附带费用估算部分

### 小额案件特殊处理

当预期收益较低（如 < 5000 元）时，额外提示：

```
⚠️ 您的案件预期收益较小（{amount} 元），建议考虑以下因素：

1. 仲裁不收费，时间成本可控 → 值得走仲裁
2. 金额虽小，但维护的是合法权益 → 不只是钱的问题
3. 协商可能是最高效的选择 → 节省时间和精力
4. 可以先协商，协商不成再仲裁 → 灵活应对

建议：{小额案件推荐路径}
```

## 五、免责与提示

每次计算输出末尾需附带：

```
───

以上计算基于您提供的信息和当前分析结果，实际结果可能因以下因素变化：
• 仲裁庭/法院的具体裁量
• 公司提交的证据和答辩
• 法律法规的更新
• 调解中的让步

本计算仅供参考，不构成法律建议。如涉及较大金额，建议咨询专业律师。
```
